| Vault filesystem -> lifecycle/movement tools | Local files must be moved, copied, removed, or archived without escaping the vault. |

| Plugin manager -> document tools | Plugin ownership and propagation behavior must remain stable after imports move. |

| T-153-06 | Tampering | `copy_document` / `move_document` move | mitigate | Preserve identity reset/preservation rules and destination containment checks; run copy/move scenarios. |
| T-153-07 | Information Disclosure | logging during moved handlers | mitigate | Preserve existing log intent and avoid adding document body, raw payload, or embedding vectors to logs. |
| T-153-08 | Elevation of Privilege | plugin ownership/import structure | mitigate | Keep plugin ownership checks and propagation intact while preventing shared-helper imports that recreate the cycle cluster. |
</threat_model>
