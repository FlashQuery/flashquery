|----------|-------------|
| MCP caller -> `search_records` | Untrusted query/filter parameters enter the records tool. |
| embedding provider -> semantic SQL | Generated vectors are sensitive operational data and must not be logged. |
| database -> logger | Query results and failures cross into observability output. |
| logger -> operator output | Logs may be persisted or read outside the MCP call context. |
