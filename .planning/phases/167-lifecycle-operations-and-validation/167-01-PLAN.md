## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| MCP caller to maintain_vault | Untrusted tool input controls lifecycle action names, scope, confirm, max_rows, and job ids. |
| Service to PostgreSQL | Lifecycle jobs and locks persist in database state and control destructive or long-running work. |
| Status/abort lookup | Caller-provided job_id crosses into persisted job lookup. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
