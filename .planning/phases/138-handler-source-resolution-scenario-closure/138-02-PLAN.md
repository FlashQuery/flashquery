## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| source_ref string to vault resolver | Untrusted references must stay inside standard document resolution. |
| markdown document to macro engine | Only selected `fqm` block source should execute. |
| archived lifecycle state to public handler | Archived macro libraries must not be executable by reference. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-138-02-01 | T | source_ref resolver | mitigate | Use `resolveDocumentIdentifier`; do not manually join untrusted paths except through resolver output. |
| T-138-02-02 | E | macro source extraction | mitigate | Use `extractMacroFences` and `selectMacroSourceBlock`; do not execute arbitrary markdown body. |
| T-138-02-03 | I | archived docs | mitigate | Frontmatter status archived maps to canonical `not_found`. |
| T-138-02-04 | D | large/malformed docs | mitigate | Parse and extraction errors return expected envelopes before evaluator invocation. |
</threat_model>
