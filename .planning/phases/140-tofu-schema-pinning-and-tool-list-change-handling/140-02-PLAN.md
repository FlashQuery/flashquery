| T-140-05 | Elevation of Privilege | index sink / registry | mitigate | Remove changed/removed tools synchronously from registry and index sink. |
| T-140-06 | Information Disclosure | reverse-request audit | mitigate | Preserve minimal audit fields and continue excluding raw request payloads. |
| T-140-SC | Tampering | npm installs | accept | No package installs. |
