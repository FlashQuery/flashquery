| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|

| T-115-02 | Tampering | `PurposeDefaultsSchema` | mitigate | Validate known loop guardrail keys by type while preserving provider pass-through fields. |
| T-115-03 | Elevation of privilege | `ModelSchema` | mitigate | Remove string-array behavioral `capabilities`; structured booleans become the only admission surface. |
| T-115-04 | Information disclosure | `flashquery.example.yml` | accept | Example config contains env refs only and no secrets; acceptance grep rejects literal legacy behavioral capability arrays. |
