## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-142-04 | Elevation of Privilege | host brokered dispatch | mitigate | Re-check `Broker.listToolsForConsumer({kind:'host', ...})` inside each callback; hidden guessed registry keys return `isError:true`. |
| T-142-05 | Tampering | schema drift handling | mitigate | Mirror Phase 140 same-server drift bundling through `broker.getPendingSchemaDrift(ctx)` before unknown-tool responses. |
| T-142-06 | Repudiation | host `tool_calls` trace | mitigate | Record host brokered calls only after returned `CallToolResult`, using request/session trace ID and resolved cost. |
| T-142-07 | Information Disclosure | broker errors | mitigate | Do not serialize normalized `raw` errors or stderr buffers into successful host responses; thrown failures use sanitized text. |
| T-142-SC | Tampering | package installs | accept | No package-manager install is planned. |
</threat_model>
