| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|

| T-124-02 | Tampering/Spoofing | reserved frontmatter | accept | `insert_in_doc` preserves existing frontmatter and does not accept frontmatter input. |
| T-124-03 | Tampering | identifier and heading mutation | mitigate | Resolve exactly one document; ambiguous heading matches return `ambiguous_identifier` instead of guessing. |
| T-124-04 | Denial of Service/Race | document write lock | mitigate | Preserve existing document lock path and map contention to canonical `conflict` where locking is enabled. |
