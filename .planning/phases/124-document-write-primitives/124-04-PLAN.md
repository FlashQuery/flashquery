| T-124-04 | Denial of Service/Race | write contention | mitigate | Preserve per-document write path and avoid deduping duplicate targets so order remains explicit. |
| T-124-05 | Information Disclosure/Access Control | disabled memory leakage | mitigate | Disabled memory targets return per-element `unsupported` without exposing memory data; document targets still succeed. |
