<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| MCP server instance -> task registry | Registry must be scoped to a registered macro tool instance, not a process singleton. |
| MCP session identity -> registry operations | List/get/cancel must filter by explicit session identity. |
| Terminal transition -> registry storage | Terminal records must not remain enumerable after completion/failure/cancellation. |
