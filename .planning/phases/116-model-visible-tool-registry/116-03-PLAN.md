## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-116-09 | Spoofing | purpose tool names | mitigate | Reject unknown tier strings and unknown named native tools with explicit config errors. |
| T-116-10 | Elevation of Privilege | `excluded_tools` without base set | mitigate | Reject `excluded_tools` unless `tools` is present, preventing ambiguous broad exposure assumptions. |
| T-116-11 | Repudiation | hard-excluded declarations | mitigate | Permit hard-excluded names through validation so registry diagnostics can identify exact removed tools. |
| T-116-12 | Denial of Service | invalid config | accept | Invalid config fails at startup with actionable errors; this is expected operator feedback, not a runtime crash. |
</threat_model>
