<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| CLI args to config lookup | Untrusted server ID selects a configured MCP server entry. |
| FlashQuery to MCP subprocess | Configured command/args spawn an external stdio server. |
| MCP subprocess stderr to terminal | Potentially sensitive stderr crosses back to the operator. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-143-01-CLI | Elevation of Privilege | `src/services/mcp-broker/cli.ts` | mitigate | Validate server ID against loaded config and pass only configured `command`/`args` to `BrokerClient`; never concatenate CLI args into a shell command. |
| T-143-02-STDERR | Information Disclosure | `BrokerClient.stderrText` failure output | mitigate | Surface stderr only on failure, rely on existing bounded stderr buffer, and keep successful stdout clean for YAML. |
| T-143-03-CONFUSION | Spoofing | Diagnostic output | mitigate | Output the selected configured server ID and exact discovered tool names; do not infer hidden consumer visibility. |
| T-143-SC | Tampering | npm installs | accept | No package-manager installs are planned; Package Legitimacy Audit says no new packages. |
</threat_model>
