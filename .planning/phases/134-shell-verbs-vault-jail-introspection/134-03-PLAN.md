| macro evaluator -> ShellJS | Evaluated macro args enter ShellJS-backed file read commands. |
| pipeline stage -> next stage | Prior command output becomes stdin for the next builtin call. |
| concurrent invocations -> process runtime | Multiple macro calls run in one Node process and must not share cwd state. |
