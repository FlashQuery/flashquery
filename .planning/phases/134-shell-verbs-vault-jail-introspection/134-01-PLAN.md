## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| macro source -> host filesystem path | Untrusted macro path strings are converted into absolute host paths. |
| host filesystem path -> macro result | Host paths returned by shell helpers are converted back to macro-visible vault paths. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-134-01 | Tampering | `resolveMacroPath` | mitigate | Normalize and resolve paths, then enforce root-or-descendant containment before any caller can use a host path. |
| T-134-02 | Information Disclosure | `resolveMacroPath` | mitigate | Throw expected `forbidden_path` for `..` and sibling-prefix escapes so macro authors cannot read outside the vault. |
| T-134-03 | Spoofing | `toMacroPath` | mitigate | Reject host paths outside the vault instead of formatting them as macro paths. |
</threat_model>
